# 快速入门

本指南帮助您快速开始 SAST Readium 开发。

## 启动开发

### Web 开发

```bash
pnpm dev
```

这将在 [http://localhost:3000](http://localhost:3000) 启动 Next.js 开发服务器，具有：

- 热模块替换 (HMR)
- React 组件快速刷新
- 错误覆盖层
- TypeScript 类型检查

### 桌面开发

```bash
pnpm tauri dev
```

此命令将：

1. 启动 Next.js 开发服务器
2. 编译 Rust 后端
3. 启动 Tauri 桌面应用
4. 为前端和 Rust 代码启用热重载

## 打开 PDF

1. 启动应用（Web 或桌面）
2. 您将看到欢迎页面，可以：
   - **拖放** PDF 文件
   - 点击 **打开文件** 浏览
   - 从 **最近文件** 中选择

!!! tip "压缩包支持"
SAST Readium 还可以从 ZIP、CBZ、RAR 和 CBR 压缩包中打开 PDF。

## 基本导航

### 键盘快捷键

| 快捷键                           | 操作            |
| -------------------------------- | --------------- |
| ++arrow-left++ / ++arrow-right++ | 上一页/下一页   |
| ++home++ / ++end++               | 第一页/最后一页 |
| ++plus++ / ++minus++             | 放大/缩小       |
| ++0++                            | 重置缩放        |
| ++f++                            | 切换全屏        |
| ++ctrl+f++                       | 搜索            |
| ++ctrl+b++                       | 切换书签        |
| ++question++                     | 显示所有快捷键  |

### 查看模式

使用工具栏切换查看模式：

- **单页**：一次显示一页
- **连续**：滚动浏览所有页面
- **双页**：并排显示对开页

### 缩放选项

- **自定义**：手动缩放（50%-500%）
- **适应宽度**：缩放页面以填充视口宽度
- **适应页面**：缩放整个页面以适应视口

## 使用注释

### 高亮文本

1. 在 PDF 中选择文本
2. 在上下文菜单中点击高亮颜色
3. 高亮自动保存

### 添加评论

1. 在注释工具栏中点击评论工具
2. 在页面上点击要添加评论的位置
3. 输入评论内容，点击外部保存

### 绘图

1. 选择绘图工具
2. 选择笔触颜色和宽度
3. 在页面上自由绘制
4. 再次点击绘图工具退出绘图模式

### 撤销/重做

- ++ctrl+z++ - 撤销上一个注释操作
- ++ctrl+y++ - 重做已撤销的操作

## AI 助手

SAST Readium 包含用于 PDF 分析的集成 AI 助手。

### 打开 AI 侧边栏

点击工具栏中的 AI 图标或按 ++ctrl+shift+a++ 打开 AI 侧边栏。

### 功能

- **PDF 上下文感知**：AI 理解当前 PDF 内容
- **工具调用**：AI 可以执行搜索、导航等操作
- **多提供商支持**：OpenAI、Anthropic 和自定义提供商
- **MCP 协议**：通过模型上下文协议连接外部工具

### 配置

1. 在侧边栏中打开 AI 设置
2. 选择您首选的 AI 提供商
3. 输入您的 API 密钥
4. 自定义模型和参数

## 自定义

### 主题

使用工具栏中的主题切换器切换主题：

- **明亮**：默认明亮主题
- **暗黑**：适合低光环境的暗黑模式
- **棕褐色**：舒适阅读的暖色调
- **自动**：跟随系统偏好

### 自定义主题

创建自定义颜色主题：

1. 打开设置 → 自定义主题
2. 点击"创建新主题"
3. 使用颜色选择器自定义颜色
4. 保存并应用您的主题

### 语言

在英语和中文之间切换：

1. 点击工具栏中的语言切换器
2. 选择您首选的语言
3. 界面立即更新

## 生产构建

### Web 构建

```bash
pnpm build
```

输出生成在 `out/` 目录，可用于静态托管。

### 桌面构建

```bash
pnpm tauri build
```

平台特定安装程序生成在：

- **Windows**: `src-tauri/target/release/bundle/msi/`
- **macOS**: `src-tauri/target/release/bundle/dmg/`
- **Linux**: `src-tauri/target/release/bundle/appimage/`
