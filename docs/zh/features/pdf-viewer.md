# PDF 查看器

PDF 查看器是 SAST Readium 的核心组件，提供丰富的阅读体验，包括多种查看模式、导航选项和自定义设置。

## 查看模式

### 单页模式

一次显示一页，适合专注阅读。

- 使用箭头键或页面按钮导航
- 简洁、无干扰的视图
- 最适合演示或详细阅读

### 连续滚动模式

所有页面在连续可滚动视图中显示。

- 自然的滚动体验
- 快速浏览长文档
- 基于可见性的页面跟踪
- 通过虚拟滚动优化性能

### 双页（对开）模式

并排显示页面，模拟实体书籍。

- 左右页面一起显示
- 适合为打印设计的文档
- 保持页面对开关系

## 导航

### 页面导航

| 方法         | 描述                                           |
| ------------ | ---------------------------------------------- |
| **箭头键**   | ++arrow-left++ / ++arrow-right++ 上一页/下一页 |
| **页面按钮** | 点击工具栏按钮                                 |
| **页码输入** | 输入特定页码                                   |
| **缩略图**   | 点击缩略图跳转到页面                           |
| **大纲**     | 点击目录条目                                   |
| **键盘**     | ++home++ / ++end++ 第一页/最后一页             |

### 跳转到页面对话框

按 ++g++ 或点击页码指示器打开跳转对话框：

1. 输入目标页码
2. 按 ++enter++ 或点击"跳转"
3. 查看器跳转到该页面

### 缩略图导航

缩略图侧边栏提供可视化导航：

- **点击** 缩略图导航到该页面
- **拖放** 重新排序页面（虚拟重排）
- **滚动** 浏览缩略图快速概览

## 缩放控制

### 缩放范围

- **最小**: 50%
- **最大**: 500%
- **默认**: 适应宽度

### 缩放方法

| 方法           | 操作                    |
| -------------- | ----------------------- |
| **工具栏按钮** | 点击 +/- 按钮           |
| **键盘**       | ++plus++ / ++minus++ 键 |
| **鼠标滚轮**   | ++ctrl++ + 滚动         |
| **捏合手势**   | 触摸设备上双指捏合      |
| **预设**       | 从下拉菜单选择          |

### 适应模式

- **适应宽度**: 缩放页面以填充视口宽度
- **适应页面**: 缩放整个页面以适应视口
- **自定义**: 手动缩放百分比

## 页面旋转

为扫描角度错误的文档旋转页面：

- **顺时针 90°**: 点击旋转按钮或按 ++r++
- **旋转持久化**: 每页旋转会被记住
- **支持**: 0°、90°、180°、270°

## 搜索

### 全文搜索

1. 按 ++ctrl+f++ 或点击搜索图标
2. 输入搜索词
3. 结果在所有页面中高亮显示
4. 使用 ++enter++ / ++shift+enter++ 在结果间导航

### 搜索选项

- **区分大小写**: 切换精确大小写匹配
- **结果计数**: 显示找到的总匹配数
- **当前位置**: 指示当前聚焦的结果

## 文本选择

### 选择文本

1. 点击并拖动选择文本
2. 选中的文本会高亮显示
3. 上下文菜单出现并提供选项

### 选择操作

- **复制**: 复制文本到剪贴板
- **高亮**: 添加高亮注释
- **搜索**: 搜索选中的文本
- **朗读**: 大声朗读选中的文本 (TTS)

## 侧边栏

### 侧边栏面板

| 面板       | 内容           |
| ---------- | -------------- |
| **缩略图** | 页面预览       |
| **大纲**   | 目录           |
| **书签**   | 用户创建的书签 |
| **注释**   | 所有注释列表   |

### 侧边栏控制

- **切换**: 点击侧边栏按钮或按 ++ctrl+b++
- **调整大小**: 拖动侧边栏边缘
- **切换面板**: 点击面板标签

## 设置

### 显示设置

- **主题**: 明亮、暗黑、棕褐色、自动
- **查看模式**: 单页、连续、双页
- **显示注释**: 切换注释可见性
- **显示页码**: 切换页码覆盖层

### 滚动设置

- **滚动灵敏度**: 调整滚动速度
- **平滑滚动**: 启用/禁用平滑滚动
- **反转滚轮**: 反转滚动方向

### 水印设置

- **水印文本**: 自定义文本覆盖
- **颜色**: 水印颜色
- **透明度**: 透明度级别
- **大小**: 文本大小
- **间距**: 水印之间的间距
- **旋转**: 水印角度

## 键盘快捷键

| 快捷键                           | 操作            |
| -------------------------------- | --------------- |
| ++arrow-left++ / ++arrow-right++ | 上一页/下一页   |
| ++home++ / ++end++               | 第一页/最后一页 |
| ++plus++ / ++minus++             | 放大/缩小       |
| ++0++                            | 重置缩放        |
| ++f++                            | 切换全屏        |
| ++ctrl+f++                       | 打开搜索        |
| ++ctrl+b++                       | 切换侧边栏      |
| ++r++                            | 旋转页面        |
| ++g++                            | 跳转到页面      |
| ++question++                     | 显示所有快捷键  |

## 触摸手势

在触摸设备上：

| 手势          | 操作          |
| ------------- | ------------- |
| **左/右滑动** | 下一页/上一页 |
| **捏合**      | 放大/缩小     |
| **双击**      | 切换缩放      |
| **长按**      | 上下文菜单    |

## 性能

### 优化

- **虚拟滚动**: 仅渲染可见页面
- **延迟加载**: 页面按需加载
- **Canvas 缓存**: 已渲染页面被缓存
- **Web Workers**: 重型操作在后台运行

### 大型文档

对于页数较多的文档：

- 缩略图渐进加载
- 搜索在后台建立索引
- 内存自动管理

## 文件操作

### 仅桌面端

在 Tauri 桌面应用中：

- **在 Finder/资源管理器中显示**: 显示文件位置
- **重命名**: 重命名 PDF 文件
- **删除**: 将文件移到回收站

### 属性

查看 PDF 元数据：

- 标题、作者、主题
- 创建和修改日期
- PDF 版本
- 文件大小
- 页数

### 元数据编辑

编辑 PDF 元数据（桌面端）：

1. 打开属性对话框
2. 点击"编辑"按钮
3. 修改字段
4. 点击"保存"

## 最近文件

快速访问最近打开的文件：

1. 在欢迎页面点击"最近文件"
2. 或使用 文件 → 最近文件 菜单
3. 点击任意文件打开
4. 如需要可清除历史记录
