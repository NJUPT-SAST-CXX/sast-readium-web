# 键盘快捷键

SAST Readium 提供全面的键盘快捷键，实现高效的导航和控制。

## 查看快捷键对话框

随时按 ++question++ 查看所有可用的键盘快捷键。

## 导航

### 页面导航

| 快捷键          | 操作             |
| --------------- | ---------------- |
| ++arrow-left++  | 上一页           |
| ++arrow-right++ | 下一页           |
| ++arrow-up++    | 向上滚动         |
| ++arrow-down++  | 向下滚动         |
| ++home++        | 第一页           |
| ++end++         | 最后一页         |
| ++page-up++     | 上一页（替代）   |
| ++page-down++   | 下一页（替代）   |
| ++g++           | 跳转到页面对话框 |

### 滚动

| 快捷键          | 操作         |

## 修饰键

### 平台差异

| Windows/Linux | macOS      | 操作         |
| ------------- | ---------- | ------------ |
| ++ctrl++      | ++cmd++    | 命令修饰符   |
| ++alt++       | ++option++ | 选项修饰符   |
| ++shift++     | ++shift++  | Shift 修饰符 |

### 组合示例

- ++ctrl+shift+s++ = 按住 Ctrl，按住 Shift，按 S
- ++ctrl+alt+f++ = 按住 Ctrl，按住 Alt，按 F

## 自定义

### 当前限制

键盘快捷键目前不可自定义。此功能计划在未来版本中实现。

### 冲突解决

如果快捷键与浏览器快捷键冲突：

1. 使用可用的替代快捷键
2. 考虑使用桌面应用
3. 检查浏览器扩展冲突

## 无障碍

### 屏幕阅读器支持

- 所有快捷键都会被朗读
- 焦点指示器可见
- 提供 ARIA 标签

### 运动无障碍

- 单键快捷键可用
- 无需快速按键序列
- 所有操作都有鼠标替代方案

## 提示

### 学习快捷键

1. 从导航开始（箭头键、Home/End）
2. 添加缩放控制（+、-、0）
3. 学习搜索（Ctrl+F）
4. 掌握注释（H、C、D）

### 效率

1. 对重复任务使用键盘
2. 结合鼠标进行精确操作
3. 学习适合您工作流程的快捷键
4. 定期练习

### 故障排除

如果快捷键不起作用：

1. 确保 PDF 查看器已获得焦点
2. 检查是否有冲突的扩展
3. 尝试先在查看器中点击
4. 验证没有对话框打开
