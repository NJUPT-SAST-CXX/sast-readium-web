# Keyboard Shortcuts

SAST Readium provides comprehensive keyboard shortcuts for efficient navigation and control.

## Viewing Shortcuts Dialog

Press ++question++ at any time to view all available keyboard shortcuts.

## Navigation

### Page Navigation

| Shortcut        | Action                      |
| --------------- | --------------------------- |
| ++arrow-left++  | Previous page               |
| ++arrow-right++ | Next page                   |
| ++arrow-up++    | Scroll up                   |
| ++arrow-down++  | Scroll down                 |
| ++home++        | First page                  |
| ++end++         | Last page                   |
| ++page-up++     | Previous page (alternative) |
| ++page-down++   | Next page (alternative)     |
| ++g++           | Go to page dialog           |

### Scrolling

| Shortcut        | Action                 |
| --------------- | ---------------------- |
| ++space++       | Scroll down one screen |
| ++shift+space++ | Scroll up one screen   |
| ++ctrl+home++   | Scroll to top          |
| ++ctrl+end++    | Scroll to bottom       |

## Zoom

| Shortcut              | Action                 |
| --------------------- | ---------------------- |
| ++plus++ or ++equal++ | Zoom in                |
| ++minus++             | Zoom out               |
| ++0++                 | Reset zoom (100%)      |
| ++ctrl+plus++         | Zoom in (alternative)  |
| ++ctrl+minus++        | Zoom out (alternative) |
| ++ctrl+0++            | Fit to width           |

## Modifier Keys

### Platform Differences

| Windows/Linux | macOS      | Action           |
| ------------- | ---------- | ---------------- |
| ++ctrl++      | ++cmd++    | Command modifier |
| ++alt++       | ++option++ | Option modifier  |
| ++shift++     | ++shift++  | Shift modifier   |

### Combination Examples

- ++ctrl+shift+s++ = Hold Ctrl, hold Shift, press S
- ++ctrl+alt+f++ = Hold Ctrl, hold Alt, press F

## Customization

### Current Limitations

Keyboard shortcuts are currently not customizable. This feature is planned for a future release.

### Conflict Resolution

If shortcuts conflict with browser shortcuts:

1. Use alternative shortcuts when available
2. Consider using the desktop app
3. Check browser extension conflicts

## Accessibility

### Screen Reader Support

- All shortcuts are announced
- Focus indicators are visible
- ARIA labels provided

### Motor Accessibility

- Single-key shortcuts available
- No rapid key sequences required
- Mouse alternatives for all actions

## Tips

### Learning Shortcuts

1. Start with navigation (arrows, Home/End)
2. Add zoom controls (+, -, 0)
3. Learn search (Ctrl+F)
4. Master annotations (H, C, D)

### Efficiency

1. Use keyboard for repetitive tasks
2. Combine with mouse for precision
3. Learn shortcuts for your workflow
4. Practice regularly

### Troubleshooting

If shortcuts don't work:

1. Ensure PDF viewer is focused
2. Check for conflicting extensions
3. Try clicking in the viewer first
4. Verify no dialog is open
